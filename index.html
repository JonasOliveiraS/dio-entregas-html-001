Uso das Tag's < h1, h2, h3, h4, h5, h6 >

As tags HTML <h1> a <h6> são usadas para criar títulos e subtítulos em uma página da web, representando diferentes níveis de hierarquia. <h1> é o título principal, <h2 > é secundário, e assim por diante. Elas ajudam na organização do conteúdo, melhoram a acessibilidade e a compreensão pelos motores de busca.

Uso da Tag < p >

A tag HTML <p> é usada para criar parágrafos de texto em uma página da web. Ela separa blocos de texto, adicionando espaçamento vertical, tornando o conteúdo mais legível. É fundamental para estruturar e organizar informações de maneira clara e coesa.

Uso da Tag < mark >

A tag HTML <mark> é usada para destacar texto em uma página da web. Ela é usada para chamar a atenção do leitor para informações importantes ou termos-chave, aplicando um realce de fundo amarelo ao texto selecionado. Isso ajuda a melhorar a ênfase e a compreensão do conteúdo.

Uso da Tag < small >

A tag HTML <small> é usada para diminuir o tamanho de texto em uma página da web. Ela é útil para indicar informações menos importantes, como notas de rodapé, legendas ou textos de menor relevância, mantendo a legibilidade e hierarquia no conteúdo.

Uso da Tag < i >

A tag HTML < i > é usada para aplicar ênfase ou itálico ao texto em uma página da web.

Uso da Tag < u >

A tag HTML <u> é usada para sublinhar texto em uma página da web.

Uso da Tag < strong >

A tag HTML < strong > é usada para dar ênfase forte a um texto, indicando que é importante ou crucial. Ela geralmente é renderizada em negrito pelos navegadores, mas seu principal propósito é semântico, destacando a importância do conteúdo. É preferível ao uso de < b > que é apenas para formatação visual.

Uso da Tag < ol >

A tag HTML < ol > é usada para criar listas ordenadas em uma página da web, onde os itens são enumerados sequencialmente, geralmente com números, letras ou símbolos. É útil para representar informações que possuem uma ordem específica, como instruções passo a passo ou rankings.

Uso da Tag < ul >

A tag HTML < ul > é usada para criar listas não ordenadas em uma página da web, onde os itens não possuem uma ordem específica e são geralmente marcados com símbolos, como pontos ou círculos. É comumente usada para representar listas de itens sem uma hierarquia definida, como listas de recursos ou itens de uma lista de verificação.

Uso da Tag < li >

A tag HTML < li > é usada para definir itens de uma lista em elementos < ul > (listas não ordenadas) ou < ol > (listas ordenadas). Cada < li > representa um item individual na lista e é geralmente precedido por um marcador, como um ponto ou número. É essencial para estruturar e organizar informações em listas em uma página da web.

Uso da Tag < a >

A tag HTML < a > é usada para criar links em uma página da web. Ela permite que os usuários cliquem em texto ou elementos para navegar para outras páginas ou recursos da web. Aqui está um exemplo de uso da tag < a > com o site "https://web.dio.me/home":

Visite o site Dio.me

Neste exemplo, o texto "Visite o site Dio.me" é o texto âncora que os usuários podem clicar para serem redirecionados para a página inicial do site "Dio.me" quando clicado. O atributo href especifica o URL de destino do link.

Uso da Tag < hr >

A tag HTML < hr > é usada para criar uma linha em uma página da web, geralmente para separar seções de conteúdo visualmente. É um elemento autônomo e não possui conteúdo. O uso de < hr > ajuda a melhorar a legibilidade e a organização do conteúdo da página.

Uso da Tag < sub >

A tag HTML < sub >sub< sub > é usada para criar texto subscrito em uma página da web, onde os caracteres são posicionados ligeiramente abaixo da linha de base. É comumente usado para representar informações como índices, fórmulas químicas ou datas em formato ordinal. Por exemplo, "H₂O" seria formatado usando < sub > para o "2" como "H< sub >2< /sub >O".

Uso da Tag < sup >

A tag HTML < sup > é usada para criar texto sobrescrito em uma página da web, onde os caracteres são posicionados ligeiramente acima da linha de base. É comumente usado para representar informações como expoentes matemáticos, notas de rodapé ou datas em formato ordinal. Por exemplo, "x²" seria formatado usando < sup > para o "2" como "x< sup >2< /sup >".

Uso da Tag < blockquote >

A tag HTML < blockquote > é usada para destacar e citar blocos de texto de outras fontes em uma página da web. Ela ajuda a diferenciar as citações do restante do conteúdo e pode incluir informações sobre a origem da citação, como a autoria e a fonte. Isso é útil para fins de atribuição e formatação adequada de citações em um documento. Por exemplo:

Esta é uma citação de um autor famoso.

― Autor da Citação

Neste exemplo, < blockquote > é usado para envolver a citação, enquanto < footer > fornece informações sobre o autor da citação.

Uso da Tag < font >

A tag HTML < font > era usada para controlar a formatação de texto em páginas da web, mas foi descontinuada no HTML5 devido à preferência por estilos CSS para estilização. O CSS oferece maior flexibilidade e conformidade com padrões modernos da web, tornando-o a abordagem recomendada para formatação de fontes. Portanto, evite o uso da tag < font >` e use CSS para estilizar texto em páginas da web.

Uso da Tag < del >

A tag HTML < del > é usada para marcar texto que foi removido ou deletado em uma página da web. Ela é útil para indicar edições ou revisões em documentos online. O texto dentro da tag < del > geralmente é exibido com uma linha de corte ou é estilizado de forma a destacar que foi removido.

Uso da Tag < abbr >

A tag HTML < abbr > é usada para criar abreviaturas ou siglas em uma página da web. Ela permite que você forneça uma explicação completa ou dica de ferramenta (tooltip) para a abreviatura, tornando o conteúdo mais acessível e compreensível. Quando o usuário passa o cursor sobre a abreviatura, a dica de ferramenta é exibida, fornecendo informações adicionais.
